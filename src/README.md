A modern industrial dashboard built using **React.js** to monitor and analyze key thermal power plant performance indicators through interactive visualizations and trend analysis.

> **Note:** This public version is intended as a portfolio demonstration. It uses sample data and excludes confidential plant information.

---

# 📌 Project Overview

The APC Dashboard provides engineers and plant operators with an intuitive interface for monitoring operational KPIs, comparing design versus actual values, and analyzing historical performance trends.

The application emphasizes usability, responsive design, reusable React components, and efficient data visualization.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/sudha991/APC-Dashboard.git
```

Go to the project directory

```bash
cd APC-Dashboard
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm run dev
```
